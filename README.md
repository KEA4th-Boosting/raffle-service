<h1 align="center">🎉 Raffle Service 🎉</h1>

## 📖 Description
카카오엔터프라이즈 SW아카데미 Team Boosting 사내 휴양시설 추첨 시스템

추첨에 대한 정보를 스마트 컨트랙트에 작성하여 추첨 일자에 컨트랙트의 당첨자 선정 알고리즘이 실행됩니다.
모든 응모 내역은 컨트랙트에 기록되며, 추첨의 결과 역시 컨트랙트에 기록됩니다.

## 📦 Installation

```bash
$ git clone 

$ cd raffle-service

$ npm install 
```

## 🏃‍♂️ Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## 🧪 Test

```bash
# unit tests
$ npm run test

# test coverage
$ npm run test:cov
```
